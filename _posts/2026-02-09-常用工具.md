### 常用软件工具

| 软件名称  | 软件描述 |
|----------|----------|
|glow| markdown 渲染工具，支持在终端中渲染 markdown 文件|
|helix| 类neovim，终端文本编辑器，速度快|
|neovim| 终端文本编辑器，文字编辑功能强|
|emacs| 终端文本编辑器，功能强大|
|alacritty| 终端模拟器，支持 GPU 加速|
|trxvt| 终端模拟器，占用内存小|
|fish| 类 bash，终端 shell，支持语法高亮、自动完成、插件扩展等功能|
|macports| macos 软件包管理工具，支持安装、卸载、升级等操作|
|ripgrep | 类 grep，支持在文件中搜索指定的字符串，支持正则表达式匹配|
|lazygit| 终端 git 客户端，支持在终端中进行 git 操作|
|obsidian| 笔记管理工具，支持 Markdown 格式。搭配termux/git使用，手机+电脑保持完美同步|
|The Silver Searcher| 代码搜索神器|
